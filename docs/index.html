Personal Settings

Personal rules (@rcmdnk) Import

For HHKB ((Shift-)Esc to grave accent (tilde))

  • For HHKB (vendor id = 2131).
  • And only for terminal, vim, emacs.
  • As HHKB's ESC is place for ` (gravce accent (back quote)) and ~ (tilde) at other keyboards.
  • Better to bind Control-[ to ESC as in below setting.
Key bindings
escape grave_accent_and_tilde

iTerm2 command (Meta) keys

  • Command to Option (i.e. Meta key) for other than n, v, w.

ESC (or Control-[) sends ESC EISU ESC

  • For terminal, vim, emacs.
  • Send ESC, EISU and ESC to stop changing at IME, disable IME and change to Normal mode at Vim at once.
  • Need to disable ESC at copy mode if GNU screen is used. Add following line in your .screenrc to disable ESC at copy mode:
    bindkey -m ^[ eval ''
Key bindings
escape escape, japanese_eisuu, escape
control-open_bracket escape, japanese_eisuu, escape

Copy and Paste, ctrl-z/x/c/v to cmd-z/x/c/v

  • For other than terminal, vim, emacs.
  • Copy and Paste with control key.
Key bindings
