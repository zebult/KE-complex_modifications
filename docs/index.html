For HHKB (Esc to grave accent and tilde)

  • Only for HHKB (vendor id = 2131).
  • And only for terminal, vim, emacs.
  • As HHKB's ESC is place for them at normal keyboard.
  • Need to map Control-[ to ESC as in below setting.
Key bindings
ESC ` (Grave accent)
Shift-ESC ~ (Tilde)

iTerm2 command (Meta) keys

  • Only for iTerm2.
  • Command to Option (i.e. Meta key) for other than n, v, w.

ESC (or Control-[) send ESC EISU ESC

  • For terminal, vim, emacs
  • Send ESC, EISU and ESC to stop changing at IME, disable IME and change to Normal mode at Vim at once.
  • Need to disable ESC at copy mode if GNU screen is used.
    • Add following line in your .screenrc to disable ESC at copy mode:
      bindkey -m ^[ eval ''
  • Key bindings:
    • ESC : ESC, EISU, ESC
    • Control-[ : ESC, EISU, ESC

Copy and Paste, ctrl-z/x/c/v to cmd-z/x/c/v

  • For other than terminal, vim, emacs.
  • Copy and Paste with control key.
  • Key bindings:
    • Control-Z : Command-Z
    • Control-X : Command-X
    • Control-C : Command-C
    • Control-V : Command-V

Next/Prev by ctrl-i/o, option-left/right arrow

  • For other than terminal, vim, emacs.
  • Key bindings:
    • Control-I : Command-] (Next Page)
    • Control-O : Command-[ (Prev Page)
    • Option-Left_Arrow : Command-] (Next Page)
    • Option-Right_Arrow : Command-[ (Prev Page)

Browsers (cmd-(shift-)tab to cmd-opt-right(left))

  • For Firefox, Chrome, Safari.
  • Key bindings:
    • Command-Tab : Command-Option-Right_Arrow.
    • Command-Shift-Tab : Command-Option-Right_Arrow.
    • Control-O : Command-[ (Prev Page)
    • Option-Left_Arrow : Command-] (Next Page)
    • Option-Right_Arrow : Command-[ (Prev Page)

Microsoft Office (ctrl-h, font, superscript/subscript)

  • For Microsoft Office (Word, PowerPoint, Excel).
  • Key bindings:
    • Control-H : Backspace.
    • Control-Command-Shift-S : Change font to Symbol.
    • Control-Command-Shift-H : Change font to Helvetica.
    • Control-= (equal) : Change to superscript.
    • Control-- (hyphen) : Change to subscript.

Change cmd-ctrl-h/j/k/l to arrow keys

  • Vim like move.
  • Key bindings:
    • Command-Control-H : Left_Arrow.
    • Command-Control-J : Down_Arrow.
    • Command-Control-K : Up_Arrow.
    • Command-Control-L : Right_Arrow.

Window Actions

  • Window actions with AppleScript.
  • Need [rcmdnk/AppleScript](https://github.com/rcmdnk/AppleScript) to be installed.
  • Key bindings:
    • Command-4 : Close the frontmost window.
    • Control-4 : Close the frontmost window.
    • Option-Space : Focus on Menu.
    • Option-Period : Move frontmost window to the proper place.
    • Command-Shift-Slash : Move all windows to the proper place.
    • Control-Shift-Y : Move window left.
    • Control-Shift-U : Move window down.
    • Control-Shift-K : Move window up.
    • Control-Shift-L : Move window right.
    • Command-Control-Shift-Y : Move window to the left screen.
    • Command-Control-Shift-U : Move window to the lower screen.
    • Command-Control-Shift-K : Move window to the upper screen.
    • Command-Control-Shift-L : Move window to the right screen.
    • Command-Control-Return : Make window almost full screen.
    • Command-Shift-Return : Make window almost full screen.
    • Command-Control-Shift-Return : Make window ~0.7 of full screen.
    • Command-Shift-H : Make window left half.
    • Command-Shift-J : Make window bottom half.
    • Command-Shift-K : Make window top half.
    • Command-Shift-L : Make window right half.
    • Command-Shift-Y : Reduce window width.
    • Command-Shift-U : Increase window height.
    • Command-Shift-I : Reduce window height.
    • Command-Shift-O : Increase window width.

Mouse cursor move

  • Need [rcmdnk/AppleScript](https://github.com/rcmdnk/AppleScript) to be installed.
  • Need to enable Mouse Key from System Preferences→Accessibility→Mouse and Trackpad.
  • Key bindings:
    • Control-Shift-H : Move cursor left.
    • Control-Shift-J : Move cursor down.
    • Control-Shift-K : Move cursor up.
    • Control-Shift-L : Move cursor right.
    • Control-Shift-N : Left click.
    • Control-Shift-M : Middle click.
    • Control-Shift-P : Right click.

Control-[ to ESC

  • Key bindings:
    • Control-[ : ESC everywhere.

Others (Dock, Sleep, Delete, IME, Misison Control, Spotlight)

  • Key bindings:
    • Control-M : Control-F3 (Focus on Dock).
    • Option-L : Control-Shift-Eject (Sleep).
    • Command-D : Command-Delete (Delete selected file).
    • Control-Delete : Command-Delete (Delete selected file).
    • Control-Space : Control-Option-Space (Choose previous IME).
    • Command-Control-Shift-H : Move to the left space of Mission Control.
    • Command-Control-Shift-L : Move to the right space of Mission Control.
    • Command-Control-F : Command-Space (Launch spotlight).
